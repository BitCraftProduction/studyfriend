Important Questions for Class 12 Computer Science (C++)

Multiple Choice Questions

Choose the correct answer from the following:-

1. What is a class in object-oriented programming (OOP)?
a) A function
b) A variable
c) A data structure
d) A blueprint for creating objects

Answer: d) A blueprint for creating objects

2. Which keyword is used to create a class in C++?
a) object
b) create
c) new
d) class

Answer: d) class

3. What is the primary purpose of a constructor in a class?
a) To destroy an object
b) To initialize an object
c) To define member functions
d) To declare variables

Answer: b) To initialize an object

4. Inheritance in OOP allows a class to:
a) Acquire properties and behavior of another class
b) Hide properties and behavior of another class
c) Prevent the creation of objects
d) Define its own properties and behavior

Answer: a) Acquire properties and behavior of another class

5. What is polymorphism in OOP?
   a) The ability of a function to perform different tasks based on the object it is called with
   b) The ability to hide the implementation details of a class
   c) The process of combining data and functions into a single unit
   d) The process of declaring multiple constructors in a class

   **Answer: a) The ability of a function to perform different tasks based on the object it is called with**

6. Which OOP concept is used to restrict access to certain members of a class?
   a) Inheritance
   b) Polymorphism
   c) Encapsulation
   d) Abstraction

   **Answer: c) Encapsulation**

7. What does the term "abstraction" mean in OOP?
   a) Showing only essential details while hiding unnecessary information
   b) Combining data and functions into a single unit
   c) Defining properties and behavior of a class
   d) Acquiring properties and behavior of another class

   **Answer: a) Showing only essential details while hiding unnecessary information**

8. Which OOP concept emphasizes the bundling of data with the methods that operate on the data?
   a) Inheritance
   b) Polymorphism
   c) Encapsulation
   d) Abstraction

   **Answer: c) Encapsulation**

9. What is the process of creating a new class from an existing class called?
   a) Encapsulation
   b) Inheritance
   c) Polymorphism
   d) Abstraction

   **Answer: b) Inheritance**

10. What is the primary benefit of using OOP concepts like classes and objects?
    a) It allows for more efficient memory management
    b) It provides a way to organize and structure complex code
    c) It simplifies the syntax of the programming language
    d) It enables faster execution of programs

    **Answer: b) It provides a way to organize and structure complex code**
