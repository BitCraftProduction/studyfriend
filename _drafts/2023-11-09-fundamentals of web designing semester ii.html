Unit I. Introduction to Computer

Computer is an electronic device that revolutionized the world. It processes data to perform various tasks. It can be programmed to execute a wide range of operations as well as, it is capable to perform complex calculations at high speeds. Nowadays, computers can be found in various forms, including personal computers, laptops, tablets, smartphones, smart watches and so on. A modern age student can not imagine life without computers and internet. In short, computers play a big role directly or indirectly on our daily lives.

Basic Components of a Computer

1. Central Processing Unit (CPU): Also known as the processor, the CPU is the brain of the computer that follows the instructions and processes the data.

2. Random Access Memory (RAM): RAM is a temporary storage area that the computer uses to store data and programs that are currently in use. It allows for quick access to data, which makes the computer run faster.

3. Storage Devices: Computers use various storage devices to store data and programs to long-term use. Some of them are hard disk drives (HDDs), solid-state drives (SSDs), and USB flash drives.

4. Input Devices: These are used to enter data and commands into the computer. Some common input devices are keyboards, mouse, touchscreens, microphones, etc.

5. Output Devices: Output devices display or present the processed information to the user. Examples - monitors, printers, speakers, projectors, etc.

6. Motherboard: This is the main circuit board of the computer that connects all the components, and it allows them to communicate with each other.

7. Operating System (OS): The operating system is the software that manages computer hardware and software resources.
